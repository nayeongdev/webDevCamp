HTML & CSS Basics Summary

HTML 요약

HTML(HyperText Markup Language)는 페이지 내용, 구조, 의미를 정의하는데 사용한다.
꾸미는 목적으로 사용하면 안된다. 꾸미는 것은 CSS를 사용한다.

  • HTML은 내용을 설명하기위해 "요소"를 사용한다.
  • HTML 요소에는 일반적으로 시작태그, 내용, 닫는 태그가 있다.
  • 이미지와 같은 빈 요소를 가질 수도 있다.
  • 속성으로 요소를 구성할 수도 있다.
  • 사용 가능한 요소들이 많이 있지만 시간이 지남에 따라 경험을 쌓을 수 있다.

CSS 요약

CSS(Cascading Style Sheets)는 페이지를 꾸미는 것에 사용한다.

  • 스타일은 속성-값 쌍을 통해 할당된다.
  • style 속성을 통해 스타일을 할당할 수 있다.
  • 코드 중복을 피하기위해, 일반적으로 전역 스타일(예:style 요소)을 사용한다.
  • 또는 외부 스타일스트 파일로 작업할 수 있다.
  • CSS로 작업할때, "상속", "특수성", "box 모델"과 같은 개념을 이해해야 한다.
